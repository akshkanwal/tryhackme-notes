# Pre-Security Learning Path

**Platform:** TryHackMe  
**Status:** In Progress

The Pre-Security path covers the foundational knowledge required for cybersecurity work: networking concepts, Linux, Windows, and basic web technologies. It is designed to build the technical baseline needed before moving into offensive or defensive security specializations.

## Modules

### Networking
| Room | Notes |
|---|---|
| [Networking Concepts](./networking/01-networking-concepts.md) | OSI model, physical through application layer |
| [Networking Essentials](./networking/02-networking-essentials.md) | DHCP, ARP, NAT, ICMP |
| [Networking Core Protocols](./networking/03-networking-core-protocols.md) | DNS, HTTP, FTP, email protocols |
| [Networking Secure Protocols](./networking/04-networking-secure-protocols.md) | TLS, SSH, VPN, CIA triad |
| [Wireshark](./networking/05-wireshark.md) | Packet capture and analysis |
| [tcpdump Basics](./networking/06-tcpdump-basics.md) | CLI packet capture |
